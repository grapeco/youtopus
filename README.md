# Requirements:
## 1. Rust
## 2. yt-dlp

# How to use this programm
## You need to fill args.txt file

## Structure of file
**first line** = Path(optional)<br>
**second line** = Output media type<br>
**third line** = Format(optional)<br>

## For audio(example):
```
out
audio
opus
```

## For video(example):
```
out
video
399+251("" for default and "custom" for choosing format)
```

## This programm working only with args.txt. You don't have choise
